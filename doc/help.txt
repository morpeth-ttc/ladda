NAME
  ladda - simple ladder tournament generator

SYNOPSIS
  ladda command ...

DESCRIPTION
  ladda is a command-line tool to maintain a simple ladder tournament, with a webpage generator.
  Add players and match outcomes, and ladda will maintain a current ranking for your players
  along with a recent history of results for each player. Data is saved in a json file.
  You can export the current state of the ladder to a static web page.

COMMANDS
  d
    Dump db to screen.
  
  e
    Export db to web in /public/index.html.
  
  m winner loser
    Add a match outcome.
  
  h
    Display this help text.

  i
    Initialize the data file.
  
  p player
    Add a player.
